An interactive data visualization dashboard showcasing the growth and impact of the Superteam Nigeria crypto ecosystem throughout 2025.

- **Virtual Events**: Online event participation
- **Contributors**: Active community contributors
- **Members**: Core community members
- **Events Hosted**: Total number of community events organized
- **Jobs Awarded**: Employment opportunities created
- **Contracts Awarded**: Professional engagements facilitated
- **Milestone Events**: Major achievements and recognition moments

## ğŸ“Š Data Updates

The dashboard currently uses static data extracted from Superteam Nigeria's monthly reports. To update the data:

1. Extract new metrics from the latest Substack report
2. Update the `monthlyData` array in the component
3. Add new milestone events to the `events` array if applicable
4. Rebuild and redeploy
